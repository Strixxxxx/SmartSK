Looking at your evidence, I can see the core issue clearly. Your logs show the classic Azure App Service node_modules extraction problem, despite your GitHub Actions packaging correctly. Here's what's happening and the complete solution:

Root Cause Analysis
GitHub Actions is working correctly - Your package contains all node_modules
Azure is running its own build process - The SCM_DO_BUILD_DURING_DEPLOYMENT: true in your logs confirms this
Azure's build is failing or incomplete - This leaves node_modules empty after deployment
Critical Issues I See:
From your logs, I notice:

SCM_DO_BUILD_DURING_DEPLOYMENT: true is appearing repeatedly
Container startup timeouts (130+ seconds)
node_modules.tar.gz exists but node_modules/ is empty
Your app is crashing with module not found errors
Complete Solution
1. Update Your GitHub Actions Workflow
Replace your current workflow with this fixed version:

yaml
# Docs for the Azure Web Apps Deploy action: https://github.com/Azure/webapps-deploy
# More GitHub Actions for Azure: https://github.com/Azure/actions

name: Build and Deploy Node.js app to Azure Web App - smartSK-backend

on:
  push:
    branches:
      - main
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    permissions:
      contents: read

    steps:
      - uses: actions/checkout@v4

      - name: Set up Node.js version
        uses: actions/setup-node@v4
        with:
          node-version: '22.x'
          cache: 'npm'
          cache-dependency-path: './backend/package-lock.json'

      - name: Clean install dependencies
        working-directory: ./backend
        run: |
          npm ci --production --no-optional
          npm list --depth=0

      - name: Verify Azure Storage Blob installation
        working-directory: ./backend
        run: |
          echo "Checking @azure/storage-blob installation..."
          npm list @azure/storage-blob
          ls -la node_modules/@azure/ || echo "No @azure modules found"

      - name: Remove unnecessary files
        working-directory: ./backend
        run: |
          find node_modules -name "*.md" -delete 2>/dev/null || true
          find node_modules -name "test*" -type d -exec rm -rf {} + 2>/dev/null || true
          find node_modules -name "*.map" -delete 2>/dev/null || true

      - name: Upload backend artifacts
        uses: actions/upload-artifact@v4
        with:
          name: backend-app
          path: ./backend
          compression-level: 6

  deploy:
    runs-on: ubuntu-latest
    needs: build
    permissions:
      id-token: write
      contents: read

    steps:
      - name: Download backend artifacts
        uses: actions/download-artifact@v4
        with:
          name: backend-app
          path: ./backend

      - name: Verify node_modules before deploy
        run: |
          echo "Verifying node_modules structure..."
          ls -la ./backend/
          ls -la ./backend/node_modules/ | head -10
          echo "Azure storage modules:"
          ls -la ./backend/node_modules/@azure/ || echo "No @azure modules"

      - name: Login to Azure
        uses: azure/login@v2
        with:
          client-id: ${{ secrets.AZUREAPPSERVICE_CLIENTID_F3C59CAD920D4D2488E5081F5E02AD69 }}
          tenant-id: ${{ secrets.AZUREAPPSERVICE_TENANTID_86BBADAEAA6540C697C5CF8E2ABC0EE1 }}
          subscription-id: ${{ secrets.AZUREAPPSERVICE_SUBSCRIPTIONID_D1E9DCD73D244760B3990633D375E2AE }}

      - name: Deploy to Azure Web App
        id: deploy-to-webapp
        uses: azure/webapps-deploy@v3
        with:
          app-name: 'smartSK-backend'
          slot-name: 'Production'
          package: ./backend

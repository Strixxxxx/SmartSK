# AI Development Prompt: Redesign Project List Module

## Overview
I need to redesign the **Project List Module** (homepage) to match the design and layout of the **Dashboard Feed Module** (client's dashboard). The goal is to create visual consistency across both pages.

## Current State Analysis

### Screenshot 1: Project List Module (NEEDS FIXING)
**Location:** Homepage (`localhost:5173/project-list`)
**Current Issues:**
- Full-width layout with minimal styling
- Images displayed in a single horizontal row
- No card container wrapping the post
- Title and author info appear directly on the page background
- Lacks visual hierarchy and organization
- No clear separation between posts
- Images are in-line without proper grid structure

### Screenshot 2: Dashboard Feed Module (REFERENCE DESIGN)
**Location:** Client Dashboard (`localhost:5173/dashboard`)
**Design Elements to Replicate:**
- Clean white card container with shadow/border
- Compact, organized layout
- Post title as a heading inside the card
- Author name styled as subtle secondary text ("By: Jeff Aldreich S. Bontuyan")
- Post content/description below author
- **2x2 grid layout for images** (not horizontal row)
- Rounded corners on images
- "+5" overlay on the 4th image indicating more images
- Proper padding and spacing throughout
- Card hover effects (if applicable)

## Required Changes

### 1. Card Container
Transform the post into a contained card:
- Add white background (`background: white`)
- Add subtle shadow or border for depth
- Apply border-radius for rounded corners (8-12px)
- Add padding inside the card (16-24px)
- Set max-width if needed to prevent overstretching
- Add margin between multiple posts

### 2. Content Layout Structure
```
┌─────────────────────────────────────┐
│  Post Title (h2/h3 styling)        │
│  By: Author Name (smaller, gray)    │
│  Post description text              │
│                                     │
│  ┌────────┐ ┌────────┐            │
│  │ Image 1│ │ Image 2│            │
│  └────────┘ └────────┘            │
│  ┌────────┐ ┌────────┐            │
│  │ Image 3│ │Image+5 │            │
│  └────────┘ └────────┘            │
└─────────────────────────────────────┘
```

### 3. Image Gallery Grid
**Critical Change:** Convert from horizontal row to 2x2 grid
- Use CSS Grid or Flexbox with wrap
- Display maximum 4 images in the preview
- Grid template: 2 columns × 2 rows
- Add gap between images (8-12px)
- Maintain aspect ratio (use object-fit: cover)
- Add rounded corners to images (4-8px)
- On the 4th image, overlay a "+X" indicator showing remaining image count

### 4. Typography & Styling

**Post Title:**
- Font size: 18-24px
- Font weight: 600-700 (semi-bold/bold)
- Color: Dark gray or black (#1a1a1a or similar)
- Margin bottom: 8px

**Author Line:**
- Format: "By: [Author Name]"
- Font size: 14px
- Font weight: 400 (normal)
- Color: Medium gray (#666 or #888)
- Font style: Italic (optional)
- Margin bottom: 12px

**Post Description:**
- Font size: 14-16px
- Font weight: 400
- Color: Dark gray (#333 or #444)
- Line height: 1.5
- Margin bottom: 16px

### 5. "+X More" Image Overlay
For the 4th image when there are more than 4 images:
- Add semi-transparent dark overlay (rgba(0,0,0,0.6))
- Center white text showing "+X" (where X = remaining image count)
- Font size: 24-32px
- Font weight: 600-700
- Ensure text is clearly visible

### 6. Responsive Behavior
- On mobile/tablet: Consider switching to single column or stacked layout
- Maintain 2x2 grid for images on most screen sizes
- Adjust card padding for smaller screens
